Getting Aquainted with MousePressed, an exercise.

Step 1:
Follow this mousePressed documentation to add a mousePressed function underneath your draw() function.

Step 2:
Inside the mousePressed function:
- Draw Two rectangles at any location, with a width and height of 20 pixels.
- Draw Two ellipses, at any location, with a width and height of 40 pixels.
HINT: after running your program, don't forget to click your canvas to see your shapes.

Step 3:
These shapes need some stellar colors. Inside the mousePressed function:
- Add a line of code to fill in your shapes with color. For its arguments, we will use the random funciton.
- HINT: the line of code that you are adding in this step should look like this fill(random(), random(), random());

Step 4:
Ensure that each random function takes in 2 arguments: 0 and 255.
HINT: The first random function has been completed for you. Repeat this for the other 2 random functions.
SECOND HINT: Click you canvas multiple times to see your sketch in action.

Congratulations! You have exercised the mousePressed Function.
